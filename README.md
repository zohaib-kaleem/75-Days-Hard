# 75 Days Challenge Tracker

A modern Python GUI application to track your 75 Days Challenge progress. Built with `customtkinter` and `matplotlib`.

## Features

- **Dashboard**: Track daily tasks including workouts, diet, water, reading, and photos.
- **Analytics**: Visualize your progress with charts showing consistency over the last 7 days and overall completion.
- **Persistence**: Your progress is saved locally in a SQLite database (`75days.db`).
- **Modern UI**: Clean design with support for Light and Dark modes.
- **Customization**: Track your current day and overall percentage.

## Installation

1. Ensure you have Python installed.
2. Install dependencies:
   ```bash
   pip install customtkinter matplotlib
   ```
3. Run the application:
   ```bash
   python main.py
   ```

## Challenge Rules Tracked

- 2 workouts per day (one must be outdoors).
- Follow a diet plan (no cheat meals/alcohol).
- Drink 1 gallon of water.
- Read 10 pages of a non-fiction book.
- Take a daily progress photo.
